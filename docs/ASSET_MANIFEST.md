# 📦 ASSET_MANIFEST.md
# PixelMon Journey — Asset Manifest

> Contract between art assets and the config layer (`src/journey/config/`).
> Every path below is generated by `spriteConfig.ts` or `themeConfig.ts`.
> Components never reference asset paths directly.

All paths are **relative to `src/assets/`**. The asset loader (M4/M5) resolves them.

Per `rules/cursor.md`: never block on missing art. Placeholder color blocks are
acceptable — configs already define final paths so real art drops in without
code changes.

---

## Naming Conventions

- kebab-case, lowercase, no spaces
- PNG with transparency for all art
- Sprite sheets: `sprites/<theme>/<creature-slug>-<animation>.png`
- Backgrounds: `backgrounds/<theme>/<scene-slug>/<layer>.png`
- Same pixel density everywhere; nearest-neighbor scaling only — never blur (DESIGN.md pixel rules)

---

## Sprite Sheets

Horizontal strips of fixed **64×64 px** frames (values mirrored in `spriteConfig.ts`).

| Animation | Frames | Sheet size | Loop | FPS |
|-----------|--------|------------|------|-----|
| idle | 4 | 256×64 | yes | 4 |
| walk | 6 | 384×64 | yes | 8 |
| evolution | 8 | 512×64 | no | 6 |

The "final idle" state reuses the stage-3 idle sheet.

## Backgrounds

Each scene directory contains **8 layer files**. Particles are procedural (M8)
and have no asset. Suggested sizes assume a 1920×1080 design viewport; tileable
layers repeat horizontally to cover camera travel.

| Layer | Parallax | Suggested size | Notes |
|-------|----------|----------------|-------|
| sky.png | 0.15× | 1920×1080 | Static gradient acceptable |
| clouds.png | 0.20× | 2048×512 | Tileable horizontally |
| moon.png | 0.25× | 512×512 | Sun art for morning/evening (layer id stays `moon`) |
| mountains.png | 0.35× | 2048×720 | Tileable horizontally |
| trees.png | 0.60× | 2048×720 | Tileable horizontally |
| ground.png | 1.00× | 1024×256 | Tileable horizontally |
| foreground.png | 1.20× | 1024×320 | Tileable, sparse silhouettes |
| fog.png | — | 1024×512 | Ambient drift (M8), not camera parallax |

Parallax speeds are canonical in `sceneConfig.ts` (`PARALLAX_SPEEDS`).

### Night scene directories (required)

